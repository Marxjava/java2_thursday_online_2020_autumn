Приемочное тестирование (Acceptance tests).

Приемочное тестирование выполняется на основании
набора типичных тестовых случаев и сценариев,
разработанных на основании требований к данному приложению.

Обычно acceptance тест это последовательность действий
выполняемых пользователем в приложении и приводящая к
определённому результату. Например для указания скидки
на товар в интеренет магазине пользователь может выполнить
следующие шаги:
- найти нужный товар через функциональность поиска;
- указать скидку товару;
- сделать заказ;
- убедиться, что скидка была применена.
